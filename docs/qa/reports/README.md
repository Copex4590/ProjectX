# QA Reports Directory

Store completed test reports here.

**Naming:** `PX-QA-RUN-{VERSION}-{NNN}.md`  
**Template:** [../RELEASE-TEST-REPORT-TEMPLATE.md](../RELEASE-TEST-REPORT-TEMPLATE.md)

Example:

```
docs/qa/reports/PX-QA-RUN-0.3.0-alpha-001.md
```

Reports are optional in git for Alpha cycles; recommended for Beta and stable releases.
