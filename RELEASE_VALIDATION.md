## PUBLIC RELEASE BLOCKED

The repository **cannot** ship a public Alpha download today. End users would reach the download page but receive **404** responses.

### Exact blockers (must resolve before public Alpha)

1. **Windows installer missing** — build `release/windows/ProjectX-Setup.exe` on Windows (`scripts\build_windows.bat` → `scripts\verify_windows_installer.bat`).
2. **Linux packages missing** — build `release/linux/ProjectX-0.3.0-alpha-x86_64.AppImage` and `projectx_0.3.0-alpha_amd64.deb` (`./scripts/build_linux_release.sh` → `./scripts/verify_linux_release.sh`).
3. **Website download copies missing** — run `./scripts/prepare_release.sh` after builds to populate `website/downloads/`.
4. **Checksums missing** — run `./scripts/generate_release_checksums.sh` (included in `prepare_release.sh`).
5. **Install smoke tests not completed** — Windows silent install/uninstall and Linux AppImage/`.deb` install on clean VMs.
6. **First Run Wizard + application launch not verified** — manual validation on Windows 10+ and Linux Mint 21+ after install.

### Recommended sequence to unblock

```bash
# Windows (native)
scripts\build_windows.bat
scripts\verify_windows_installer.bat

# Linux
./scripts/build_linux_release.sh
./scripts/verify_linux_release.sh

# Publish prep (Linux host, after copying Windows artifact if needed)
./scripts/prepare_release.sh
./scripts/verify_release.sh
./scripts/validate_public_download.sh
```

Re-run this validation after blockers are cleared. All **P0** items must be **PASS** and the final section must read **READY FOR PUBLIC ALPHA** before announcing the release.
