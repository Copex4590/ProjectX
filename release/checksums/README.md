# Release checksums

Auto-generated by:

```bash
./scripts/generate_release_checksums.sh
```

Or as part of:

```bash
./scripts/prepare_release.sh
```

| File | Purpose |
|------|---------|
| `SHA256SUMS` | Combined checksums for all release artifacts |
| `*.sha256` | Individual checksum file per artifact |

Verify:

```bash
./scripts/verify_release.sh
```
