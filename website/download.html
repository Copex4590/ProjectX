Which file should I download?

Linux Mint, Ubuntu, or Debian

Download ProjectX.deb — recommended. Double-click the file to open GDebi Package Installer, or run sudo dpkg -i ProjectX.deb. Project X appears in your applications menu as Project X with the description Professional Maritime Monitoring Platform. A desktop shortcut and post-install launch are enabled by default. Uninstall later from Software Manager or with sudo dpkg -r projectx.

Download ProjectX.AppImage only if you want a portable copy without installing to the system. You must mark the file executable, then run it from your file manager or terminal. It does not add a menu entry automatically.

SHA256SUMS — optional. Use it only if you want to verify that your download was not corrupted.

Windows

Download ProjectX-Setup.exe and run the installer. Optional SHA256SUMS is for integrity verification only.
