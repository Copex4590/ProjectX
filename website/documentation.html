Getting started

Project X is a desktop application for monitoring Danube vessel traffic. After installation, the first-run wizard guides you through language selection, observation point setup, AIS configuration, and optional camera setup.

  1. Download the installer for your platform from the Download page.
  2. Launch Project X and complete the first-run wizard.
  3. Open the Dashboard to verify AIS connection and live ship counts.
  4. Open the Live Map to view vessels on the interactive Leaflet map.
  5. Explore Timeline, Statistics, Alerts, and Camera features from the sidebar.

System requirements

Project X requires a modern desktop system with Qt WebEngine support for maps.

  • Windows: Windows 10 or later, 64-bit, 4 GB RAM minimum
  • Linux: Linux Mint or compatible distro with X11 or Wayland
  • Display: 1280×720 minimum resolution recommended
  • Network: Internet connection for AIS data and map tiles

Optional: mpv or vlc on PATH for external camera playback backends.

Windows installation

Run the Windows installer and follow the setup wizard. User configuration is stored under:

%APPDATA%\Project X\config\
%APPDATA%\Project X\data\

For development builds, use scripts\build_windows.bat on a native Windows system.

Linux installation

Run the Linux installer script or install from source. The desktop installer places files under:

~/.local/share/projectx

A launcher and applications menu entry are created automatically.

Build from source

Clone the repository and install Python dependencies:

git clone https://github.com/Copex4590/ProjectX.git
cd ProjectX
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cd src
python3 main.py

See BUILD_WINDOWS.md and installer/README.md in the repository for packaging instructions.
