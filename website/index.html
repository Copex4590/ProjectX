Danube vessel monitoring

Monitor vessels on the Danube with live AIS, maps, and cameras.

Project X is a desktop platform for real-time ship tracking, timeline history, statistics, alerts, and camera playback — built for operators who need a clear picture of river traffic.

Release notes

Highlights from Project X 0.3.0-alpha.

  • Vessel database with advanced search, filters, and ShipRegistry synchronization
  • Timeline recording with arrival and departure detection
  • Statistics dashboard with cached aggregates and charts
  • Alert rules engine, Alert Center, and rules management UI
  • 250 SVG vessel flags and photo framework in the vessel card
  • Camera packs, playback backends, and diagnostics tooling
  • Release packaging readiness and native Windows build pipeline

Full release notes are included in the repository under docs/RELEASE_NOTES_0.3_ALPHA.md.
